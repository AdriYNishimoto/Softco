1. Introdução

Este arquivo tem com o intuito auxiliar na execução e utilização do projeto desenvolvido. É importar seguir os passos com atenção para que não gere nenhum transtorno por parte
dos avaliadores na hora da utilização do mesmo


2. Execução do FrontEnd (Dashboard)

Para realizar poder acessar nosso projeto, siga os seguintes passos:

    1- Abra a pasta "frontend" no seu explorador de arquivos
    2- Dentro da pasta "frontend" encontre o arquivo chamado "index.html"
    3- Clique duas vezes com o botão esquerdo do mouse e abra-o no navegador de preferência

3. Dashboard

Nosso projeto é dividido em duas partes, a primeira é a landing page onde explica sobre o nosso projeto e contém o video pitch desenvolvido e a segunda é o nosso dashboard.
Para acessar o dashboard clique no botão "Login" localizado no canto superior direito do site, você será direcionado a tela para preencher os campos de E-mail e Senha, com isso
preencha o campo do email e senha seguindo este padrão:

    Preencher campo E-mail: email@email.com
    Preencher campo Senha: senha1234

Após o preenchimento prossiga apertando em "Entrar";

4. Chat

Para utilizar nosso chat você deverá seguir os seguintes passos (OBS: seguir os passos como descrito para que não haja conflitos!):

    1- Abra a pasta "backend" no seu explorador de arquivos
    2- na parte superior do seu explorador de arquivos conterá o caminho onde está essa pasta, ex: C:\Users\Desktop\SoftCo\backend. Clique neste campo e apague o caminho
    especificado e digite a palavra "cmd" e pressione "Enter".
    3- Após pressionar "Enter", o prompt de comando do windows será aberto, após aberto digite o comando "npm install" e depois "npm start" para realizar a execução do servidor, aguarde 5 segundos e 
    retorne para o Dashboard
    4- Após o servidor estar rodando, abra a aba "Chat", nela você irá se deparar com o chat aberto.
    5- Para realizar o teste inicial se o servidor está funcionando digite no campo "Mensagem..." a mensagem "Oi", se você obtiver uma resposta quer dizer que você executou 
    os passos anteriores corretamente.
    6- Após a realização do teste, você poderá testar nosso chat (Continuação no tópico 4.1).
    7- Caso tenha finalizado a simulação, retorne ao prompt de comando e aperte as teclas "CTRL + C", aparecerá uma pergunta no prompt dizendo se deseja finalizar, digite "s" e 
    aperte a tecla "ENTER".

4.1 Perguntas

Nosso chat ainda não foi integrado um modelo de IA Generativa, pois, ainda não conseguimos achar um modelo confiável que consiga gerar as respostas satisfatórias para as 
parguntas feitas e os modelos que estão disponíveis no mercado precisam de licenças para a sua utilização. Para que fosse possível a simulação de como ficará o "Chat" integrado
até o evento Next, criamos uma base com perguntas e respostas pré-definidas a partir do desafio que nós escolhemos.

    * Qual e a causa de falhas intermitentes no sistema?
    * Como resolver um problema de desempenho em um aplicativo?
    * O que fazer se um servidor está fora do ar?

    * Como instalar um novo aplicativo?

    * Como limpar o cache do navegador?
